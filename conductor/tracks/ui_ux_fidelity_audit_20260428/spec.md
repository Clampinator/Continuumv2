# Specification: UI/UX Fidelity & Readability Audit

## Objective
Polishing the visual and interactive components of the Lifeline tool to ensure it is both beautiful and functional for long-term play.

## 1. Visual Polish
*   **Label De-confliction:** Implement an algorithm to adjust the placement of overlapping event titles and date labels.
*   **High-Contrast Cues:** Enhance the visual distinction between standard events, spans, and "Yet" nodes.
*   **Aesthetic Consistency:** Ensure all colors, fonts, and borders align with the "Historical-Tech" design language.

## 2. Tooltip Enhancement
*   **Rich Context:** Tooltips should display relevant metadata (Era, Experience, Location, and Recency Bonus) in a clean, hierarchical layout.
*   **Interactive Tooltips:** Support clicking links inside tooltips (if applicable).

## 3. Spreadsheet Usability
*   **Density Management:** Allow users to toggle between "Compact" and "Comfortable" row heights.
*   **Advanced Filtering:** Support filtering by Era or Experience to make large histories manageable.
